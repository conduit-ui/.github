# Conduit UI

**Your Personal Developer API Gateway**

Component-based, AI-orchestrated developer workflows. Escape corporate tool tyranny.

## Core Packages

| Package | Purpose |
|---------|---------|
| [connector](https://github.com/conduit-ui/connector) | GitHub API transport layer |
| [issues](https://github.com/conduit-ui/issues) | GitHub Issues management |
| [prs](https://github.com/conduit-ui/prs) | Pull Request operations |
| [repos](https://github.com/conduit-ui/repos) | Repository management |
| [commits](https://github.com/conduit-ui/commits) | Commit operations |
| [actions](https://github.com/conduit-ui/actions) | GitHub Actions interface |
| [review](https://github.com/conduit-ui/review) | Code review workflows |
